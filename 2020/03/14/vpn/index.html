谷歌服务器搭建vpn

什么是VPN

我们访问不了google，是因为被zf封锁了，也就是俗称的被墙，那么如何绕过墙的限制呢？这就需要vpn了。什么是vpn？

通俗来说，你在国内访问不了google，但是假设你有一个外国的小伙伴，你们经常微信联系，外国的小伙伴是可以访问google的，那么你只需要告诉你的小伙伴你需要google什么东西，然后让外国小伙伴拍照通过微信发给你，这样你不就能看到google到的东西了吗？这个小伙伴就是你的vpn。

最近google云计算平台针对大陆有了优化政策，注册白送$300，买个服务器完全不要钱= =基本上可以白嫖一年服务器，理想情况下一年之后注销账号重新注册无限白嫖，有了这台服务器我们就可以搭建vpn啦，那么如何在google家的服务器上搞事情呢？

条件

1.需要一张visa卡，会预付¥30检测卡状态，会立刻退还，全程不花一分钱
2.翻墙状态下进行

注册流程

1.点击 google cloud platform网址

2.输入你自己的google账号
3.地区选择“台湾”，勾选服务条款

4.账号类型选择“个人”，税务信息选择“未登记的个人”，地址随便输，但是邮编必须和选择市区的邮编一致，具体可以在网上查询，这里我选择“台北市中山区”，邮编是“10441”，剩下随便瞎填就可以

5.信用卡信息填写完整，取消勾选后续自动扣款，google已经注明不会自动收费，然后选择“开始免费试用，白嫖开始

配置google服务器

1.在侧边菜单栏中选择Compute Engine，VM实例

2.选择“创建实例”

3.点击“区域”，更改成“aisa-east1台湾”，地区选择c

4.勾选“允许HTTP流量”和“允许HTTPS流量”

5.点击创建
6.选择 右侧边菜单栏-vpc网络-外部ip地址

7.将ip地址改为静态，名称随意

9.点击保留，此操作是为了防止重启的时候服务器ip地址改变

10.右侧边菜单栏-Computer engine-VM实例，记下ip地址，然后点击实例旁边的ssh

11.点击ssh，会出现如下操作窗口：

12.在命令行中输入sudo -i，回车（这步切换成root用户，即获得完全控制权）

后面所有命令行都在此窗口#后面输入即可

服务器端vpn配置

1.安装v2ray脚本

#安装v2ray脚本
bash <(curl -L -s https://install.direct/go.sh)

#查看默认配置，记下port端口号和id
cat /etc/v2ray/config.json

#开始运行v2ray
 service v2ray start

客户端vpn配置

将记下的ip地址，端口号和id填入支持vmess协议的代理软件，比如shadowsocks

右上角添加节点

填入节点信息

1.服务器填入ip地址

2.端口填入刚才记下的服务器端口

3.UUID填写刚才记下的id

其他均为默认，点击完成，打开vpn开关，打开google测试
